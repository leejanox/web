1. 구조 및 레이아웃 요소

    <header>: 페이지나 섹션의 상단을 나타내며, 주로 로고, 네비게이션, 제목 등이 포함됩니다.
    <footer>: 페이지나 섹션의 하단을 나타내며, 주로 저작권 정보, 연락처, 소셜 링크 등이 포함됩니다.
    <section>: 문서 내의 특정 주제나 섹션을 구분할 때 사용됩니다.
    <article>: 독립적으로 구분 가능한 콘텐츠 블록을 나타내며, 기사나 블로그 포스트처럼 하나의 독립적인 내용을 의미할 때 사용합니다.
    <aside>: 페이지의 주요 콘텐츠와는 별개의 부가적인 콘텐츠를 나타냅니다. 예를 들어 사이드바에 관련 링크나 광고 등이 있을 때 사용합니다.
    <nav>: 페이지 내의 주요 탐색 링크들을 그룹화할 때 사용됩니다.
    <main>: 페이지의 주요 콘텐츠를 나타내며, 페이지에서 가장 중요한 내용을 담고 있습니다. 보통 헤더, 푸터, 내비게이션 등을 제외한 주요 콘텐츠를 감쌉니다.

2. 텍스트 관련 요소

    <h1> ~ <h6>: 제목 요소로, <h1>이 가장 중요하고 큰 제목이며, <h6>은 가장 작고 덜 중요한 제목입니다.
    <strong>: 중요한 텍스트를 강조할 때 사용되며, 굵게 표시됩니다.
    <em>: 의미적으로 강조해야 하는 텍스트를 나타내며, 기울임꼴로 표시됩니다.
    <blockquote>: 인용 블록을 나타냅니다. 긴 인용문을 작성할 때 사용됩니다.
    <pre>: 서식을 유지한 채 텍스트를 나타낼 때 사용합니다. 주로 코드 블록에 활용됩니다.
    <code>: 코드 조각을 표시할 때 사용됩니다. 텍스트는 주로 고정폭 글꼴로 표시됩니다.

3. 미디어 관련 요소

    <img>: 이미지를 삽입할 때 사용합니다.
    <audio>: 오디오 파일을 삽입할 때 사용합니다.
    <video>: 비디오 파일을 삽입할 때 사용합니다.
    <figure>: 이미지, 다이어그램, 삽화 등과 같은 독립적인 콘텐츠를 포함하는 컨테이너입니다. 보통 <figcaption>과 함께 사용됩니다.
    <figcaption>: <figure> 요소에 포함된 이미지나 삽화에 대한 설명을 추가할 때 사용됩니다.

4. 리스트 요소

    <ul>: 순서가 없는 리스트를 나타내며, 목록 항목은 <li> 요소로 감쌉니다.
    <ol>: 순서가 있는 리스트를 나타내며, 목록 항목은 <li> 요소로 감쌉니다.
    <li>: 리스트 항목을 나타내며, <ul> 또는 <ol>의 자식 요소로 사용됩니다.
    <dl>: 설명 목록을 나타내며, 용어와 설명이 짝을 이루는 경우 사용됩니다.
    <dt>: 설명할 용어를 나타냅니다.
    <dd>: 용어에 대한 설명을 나타냅니다.

5. 폼 관련 요소

    <form>: 사용자 입력을 위한 폼을 정의합니다.
    <input>: 다양한 유형의 사용자 입력 필드를 정의합니다. (text, password, checkbox, radio, submit 등)
    <textarea>: 여러 줄의 텍스트 입력을 위한 필드를 정의합니다.
    <button>: 버튼을 정의하며, 폼 제출, 리셋, 일반 버튼 용도로 사용할 수 있습니다.
    <select>: 드롭다운 목록을 생성합니다.
    <option>: <select> 요소 내의 선택 항목을 정의합니다.
    <label>: 입력 요소에 대한 설명(라벨)을 제공합니다. 사용자 경험을 위해 폼 요소와 함께 사용하는 것이 좋습니다.

6. 표 관련 요소

    <table>: 표를 생성할 때 사용하는 컨테이너입니다.
    <tr>: 표의 행을 나타냅니다.
    <th>: 표의 헤더 셀을 나타내며, 일반적으로 굵은 글씨로 표시됩니다.
    <td>: 표의 데이터를 담는 셀을 나타냅니다.
    <caption>: 표에 대한 제목이나 설명을 추가합니다.
